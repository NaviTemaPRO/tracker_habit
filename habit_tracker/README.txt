- After launching the application, a menu with options will appear in the terminal:
- 1. Add a habit: Enter the name of the habit to add it to the list.
- 2. Mark the habit as completed: Enter the name of the habit to mark it as completed on the current date.
- 3. Show habits: Displays all habits and statistics on their completion.
- 4. Exit: Exit the application.

An example of working with the application

- When you select the option to add a habit, the program will ask you to enter its name.
- After that, you can mark the habit as completed by viewing all added habits and their current status.

- После запуска приложения в терминале появится меню с опциями:
  - 1. Добавить привычку: Введите название привычки, чтобы добавить её в список.
  - 2. Отметить выполнение привычки: Введите название привычки, чтобы отметить её выполнение на текущую дату.
  - 3. Показать привычки: Отобразит все привычки и статистику по их выполнению.
  - 4. Выход: Выход из приложения.

Пример работы с приложением

- Когда вы выберете опцию добавления привычки, программа попросит вас ввести её название.
- После этого вы можете отметить выполнение привычки, просмотрев все добавленные привычки и их текущий статус.
